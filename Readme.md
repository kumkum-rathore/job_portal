# 🚀 JobX - Professional Job Portal (MERN Stack)

JobX ek full-stack job portal application hai jise **MERN Stack** (MongoDB, Express, React, Node.js) ka use karke banaya gaya hai. Yeh platform Job Seekers ko naukri dhoondne aur Employers ko talent hire karne mein madad karta hai.

---

## ✨ Key Features

### For Job Seekers:
* **Advanced Search:** Jobs ko keywords, location, aur category ke hisaab se filter karein.
* **Easy Application:** Single click mein jobs ke liye apply karein.


### For Employers:
* **Job Posting:** Nayi jobs post karein aur unhe manage karein.
* **Applicant Tracking:** Dekhein kisne apply kiya hai aur unka status change karein.


## 🛠️ Tech Stack

**Frontend:** React.js, Tailwind CSS, Redux Toolkit
**Backend:** Node.js, Express.js
**Database:** MongoDB Atlas
**Authentication:** JWT (JSON Web Tokens) & Bcrypt
**Deployment:** Vercel (Frontend), Render (Backend)
